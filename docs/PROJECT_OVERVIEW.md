# Project Documentation

## Overview
This is the Sensitivity DPI Converter project - a Java Swing application for converting mouse sensitivity across competitive games.

- Real-time Sensitivity Conversion
- Modern Dark UI with Smooth Animations
- No External Dependencies
- Java 25 Compatible
